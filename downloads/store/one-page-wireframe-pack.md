# One-Page Wireframe Pack

Use these outlines to plan a small page before design or development.

## Landing Page

1. Hero
   - Offer name
   - One-sentence value proposition
   - Primary action
   - Secondary proof or trust signal

2. Problem And Outcome
   - Current pain point
   - Desired result
   - Why this offer fits

3. Offer Details
   - What is included
   - Who it is for
   - Who it is not for

4. Proof
   - Example result
   - Short testimonial
   - Relevant screenshot or metric

5. Final Action
   - Repeat primary action
   - Set expectation for what happens next

## Portfolio Page

1. Intro
   - Name
   - Role or specialty
   - Best contact action

2. Featured Work
   - 3 to 6 projects
   - Problem, role, result
   - Links or screenshots

3. Skills
   - Practical categories
   - Tools only where useful

4. About
   - Short background
   - Working style

5. Contact
   - Preferred contact method
   - Availability note

## Service Page

1. Service Summary
   - Clear service name
   - Who it helps
   - Starting price or quote note

2. Deliverables
   - Included items
   - Timeline range
   - Revision policy

3. Process
   - Intake
   - Build
   - Review
   - Delivery

4. Examples
   - Related projects
   - Screenshots or short notes

5. Start
   - Form, checkout, or contact action
