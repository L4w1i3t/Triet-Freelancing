YTDUI

A graphical user interface for simplified video downloading using yt-dlp.

What is YTDUI?

YTDUI is a clean, user-friendly desktop application that wraps around yt-dlp — the powerful command-line video downloader. Instead of memorizing terminal commands and flags, YTDUI gives you a straightforward graphical interface to download videos and audio from hundreds of supported sites. It also includes a built-in media converter so you can convert existing files between formats without leaving the app.

Features

Video & Audio

Download videos in multiple formats or extract audio only with a single click.

Quality Selection

Choose your preferred resolution and format before downloading.

Built on yt-dlp

Leverages the full power of yt-dlp with support for hundreds of sites.

Desktop App

Native desktop experience built with Python and Tkinter.

Media Converter

Convert existing files between video and audio formats directly inside the app — no separate tool needed.

Tech Stack

Python Tkinter yt-dlp ffmpeg
